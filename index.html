HTML

  • The head element contains information about the webpage
  • The body element represents the visible content shown to the user
  • The head contains "metadata" which holds information about the page that is used by the browser such as character set, discription, keywords, author, and viewport
  • Element vs Tag: element refers to the entire element from opening tag to closing tag while tag refers only to what appears inside the angle brackets
  • Elements nested inside other elements are called child elements

CSS

  • A margin indicates how much space we want around the outside of an element
  • A padding indicates how much space we want around the content inside an element
  • CSS Rule: a selector with property and value of property
  • Two or more elements that will share the same styling can be grouped togeher
  • CSS link page goes inside the head, after the last meta
  • Display:block= assigns each image to start on a new line
  • Class attributes: allow us to share a CSS rule with any element we want by assigning the rule with a class selector

Git

  • git status: checks what branch we are currently on
  • git checkout main: checks out of main branch
  • git pull origin main: pulls down latest main branch
  • git checkout -b branch-name : creates a new branch and switches to it
  • git add -A: Staging area for files to be commited, adds all changes
  • git commit -m "commit message": sends changes to repo
  • git push origin [feature/starter-code] : sends local changes to remote repsitory

JavaScript

  • A variable is a named container that allows us to store data in our code
  • Control flow is the order in which a computer executes code in a script
  • If and else: if a condition is false it will go to the second, else, condition until it finds a truthy condition
  • Arrays are used to hold a group of data and can inlcude strings, Numbers, and Booleans
  • Arrays are zero indexed, first item starts with 0
  • For Loop: contains variable x eqauling zero; x being less than array length; and increase by 1 for every iteration x++;
  • A function gives instructions and must be called to execute
